GitHub Pages是什么？
GitHub Pages 可以被认为是用户编写的、托管在github上的静态网页。由于它的空间免费稳定， 
可以用于介绍托管在github上的Project或者搭建网站。有两种形式: Project Site 和 User/Org Site，
二者之间的差异可以戳 GitHub Pages。基于 GP 创建Site是很方便的，这有一个简单的教程： 
学习 Github Page 教你分分钟搭建自己的博客
gp 生成的网站的默认域名是 username.github.io 或者 username.github.io/project-name，
但gp是支持自定义域名的：Custom Domain Name。购买域名之后，可以和默认的二级域名进行绑定，
教程参考：购买域名、设置DNS
更多关于gp的信息，可以戳：Github Pages Help
